Plugin ที่มีมาให้

  • jQuery v2.2.4 (เนื่องจาก Foundation 6 และปลั๊กอินบางตัวยังไม่รองรับ jQuery 3)
  • jQuery Mousewheel : ปลั๊กอินสำหรับจัดการ Mousewheel event รองรับ cross-browser ดูข้อมูลเพิ่มเติมได้ที่นี่
  • jQuery Easing : ปลั๊กอินเสริมสำหรับอนิเมชั่นเพิ่มเติม ใช้งานได้ดีกับ jQuery animation ดูข้อมูลเพิ่มเติมที่นี่
  • jQuery Resizestop : ใช้แทน event $(window).resize() ดูวิธีใช้งานได้ที่นี่
  • Imagesloaded : ปลั๊กอินสำหรับ detect ว่ารูปใน container นั้นๆโหลดเสร็จแล้วหรือไม่ ดูวิธีการใช้งานได้ที่นี่
  • Mixins และ functions ของ sass โดยดูรายชื่อฟังก์ชั่นได้ที่ไฟล์ src/sass/_includes.scss
    หากต้องการอ้างถึงเพื่อนำไปช่วยในการเขียน sass ให้อ้างถึงไฟล์นี้ไฟล์เดียวก็พอและต้องอ้างถึง path file ให้ถูกด้วย (ส่วนใหญ่ใช้ @import "../_includes")

ถ้าจะใช้ไฟล์นี้เป็น index และยังต้องการเก็บไฟล์นี้ไว้อ้างอิง ให้ใช้ code ในไฟล์นี้เป็น template แล้วเปลี่ยนชื่อไฟล์นี้เป็น _doc.html ก็ได้
